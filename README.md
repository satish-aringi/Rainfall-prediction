# Rainfall-prediction
Machine Learning project to predict rainfall using Random Forest, XGBoost, LightGBM, and Logistic Regression. Implemented feature engineering, SMOTE for class balancing, and a 70-20-10 stratified split. Achieved 91% test accuracy using a soft voting ensemble model.
This project builds a supervised classification model to predict rainfall occurrence using multiple ML algorithms.
